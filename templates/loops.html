All Loop Properties Explained

All examples use your 4-item list from views.py.

Example 1: Basic Loop ({{ task }})

    {% for task in tasks %}
  • {{ task }}
    • {% endfor %}

Example 2: forloop.first and forloop.last

    {% for task in tasks %} {% if forloop.first %}
  • {{ task }} (This is the first item)
    • {% elif forloop.last %}
  • {{ task }} (This is the last item)
    • {% else %}
  • {{ task }} (This is a middle item)
    • {% endif %} {% endfor %}

Example 3: forloop.counter (Starts at 1)

{% for task in tasks %}

{{ forloop.counter }}. {{ task }}

{% endfor %}

Example 4: forloop.counter0 (Starts at 0)

{% for task in tasks %}

Index {{ forloop.counter0 }} is: {{ task }}

{% endfor %}

Example 5: forloop.revcounter (Counts down to 1)

{% for task in tasks %}

{{ forloop.revcounter }} items left (including this one). Current item: {{ task }}

{% endfor %}

Example 6: forloop.revcounter0 (Counts down to 0)

{% for task in tasks %}

{{ forloop.revcounter0 }} items left (after this one). Current item: {{ task }}

{% endfor %}
